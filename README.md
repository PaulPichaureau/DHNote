# DHNote

Bibliothèque Python pour manipuler les notes du projet Dharmalib (format Obsidian.md).

## Installation

```bash
uv sync
```

## Commandes disponibles

- `dhnote-sort` — trie les en-têtes des notes
- `dhnote-update` — met à jour les métadonnées
- `dhnote-mergedir` — fusionne un répertoire de notes
